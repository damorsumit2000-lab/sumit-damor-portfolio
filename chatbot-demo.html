Before You Start

To use this chatbot demo, you need to:

  1. Get a free API key from Google AI Studio
  2. Open chatbot.js in a text editor
  3. Replace YOUR_GEMINI_API_KEY_HERE on line 7 with your actual API key
  4. Save the file and refresh this page

✅ Features

What This Chatbot Can Do

  • Answer questions about Sumit's professional experience at Groww
  • Provide details about education (MBA, BA Economics)
  • Discuss projects and technical skills
  • Share information about achievements and certificates
  • Provide contact information
  • Remember conversation context for natural dialogue

💬 Try These Sample Questions:

Tell me about his achievements

📝 Next Steps

Once you've tested the chatbot and it's working correctly:

  1. Run python3 add-chatbot-to-html.py to integrate it into your portfolio
  2. Or manually add the chatbot files to index.html
  3. Commit and push to GitHub Pages
  4. Your portfolio will now have an AI assistant!

See README_CHATBOT.md for detailed documentation.
